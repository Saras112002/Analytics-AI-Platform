# AI Analytics Platform

An AI-powered business analytics platform that ingests datasets and generates consultant-level insights using Large Language Models. Built with FastAPI, Pandas, and a provider-agnostic LLM architecture.

## Features

- **Smart File Ingestion** — Multi-encoding CSV/Excel support with automatic schema detection
- **AI-Powered Analysis** — Generates business insights using specialized AI agents
- **Provider-Agnostic LLM** — Wrapper pattern supports OpenRouter, Claude, OpenAI
- **Production Patterns** — Async endpoints, dependency injection, environment-based config
- **Auto API Documentation** — Interactive OpenAPI docs at /docs
